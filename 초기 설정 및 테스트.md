 🚀 Step 5: 초기 설정 및 테스트

  1. 초기 데이터 동기화

  Google Apps Script 편집기에서:

  1. Code.gs 파일 열기
  2. 상단 함수 선택 드롭다운에서 initialSetup 선택
  3. ▶️ 실행 버튼 클릭
  4. 권한 요청이 나타나면:
    - "권한 검토" 클릭
    - Google 계정 선택
    - "고급" 클릭 → "멘토링프로그램시스템(안전하지 않음)으로 이동" 클릭
    - "허용" 클릭

  이렇게 하면 Forms 응답 데이터가 "프로그램목록" 시트로 자동 변환됩니다.

  2. 웹앱 배포

  1. 상단 메뉴에서 배포 → 새 배포 클릭
  2. 설정:
    - 유형: 웹 앱
    - 설명: "멘토링 프로그램 시스템 v1"
    - 실행: 나
    - 액세스 권한: 모든 사용자
  3. 배포 버튼 클릭
  4. 생성된 웹 앱 URL 복사 (중요! 이 URL을 저장해두세요)

  3. 테스트

  웹 앱 URL을 브라우저에서 열어 테스트:
  - 기본: https://script.google.com/.../exec (멘티 신청 페이지)
  - 신청현황: https://script.google.com/.../exec?page=status
  - 멘토 조회: https://script.google.com/.../exec?page=mentor
  - 관리자: https://script.google.com/.../exec?page=admin

  📱 Step 6: Google Sites 설정

  1. Google Sites 생성

  1. https://sites.google.com 접속
  2. + 새 사이트 만들기 클릭
  3. 사이트 이름: "2025 신입생 멘토링 프로그램"

  2. 페이지 구성

  상단 메뉴에 4개 페이지 추가:

  페이지 1: "프로그램 신청"

  1. 우측 페이지 탭 → + 클릭
  2. 페이지 이름: "프로그램 신청"
  3. 페이지에서 삽입 → 임베드 → URL 임베드
  4. URL 입력: 웹앱URL?page=apply
  5. 전체 페이지로 설정

  페이지 2: "내 신청현황"

  1. 페이지 추가 → "내 신청현황"
  2. URL 임베드: 웹앱URL?page=status

  페이지 3: "멘토 전용"

  1. 페이지 추가 → "멘토 전용"
  2. URL 임베드: 웹앱URL?page=mentor

  페이지 4: "관리자"

  1. 페이지 추가 → "관리자"
  2. URL 임베드: 웹앱URL?page=admin

  3. Sites 게시

  1. 우측 상단 게시 클릭
  2. 웹 주소 설정 (예: mentoring2025)
  3. 게시 클릭

  ✅ 완료 체크리스트

  - Google Sheets 4개 시트 생성
  - Google Apps Script 프로젝트 생성
  - 코드 파일 복사 (Code.gs, DataManager.gs, Validation.gs, Admin.gs)
  - HTML 파일 복사 (styles.html, index.html, status.html, mentor.html, admin.html)
  - initialSetup 함수 실행
  - 웹앱 배포
  - Google Sites 설정
