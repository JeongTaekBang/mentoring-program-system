# 멘토링 프로그램 시스템 구현 계획서

## 1. 시스템 개요
대학교 신입생 멘토링 프로그램의 신청 및 관리를 위한 Google 기반 시스템

### 주요 일정
- **멘토 프로그램 등록**: 2025년 9월 5일까지
- **멘티 프로그램 신청**: 2025년 9월 8일 ~ 9월 12일
- **프로그램 운영**: 2025년 9월 22일 ~ 12월 12일 (평일만)

### 시스템 요구사항
- 멘토: 105명, 프로그램 개설 (총 3시간, 1시간 단위)
- 멘티: 멘토들이 개설한 프로그램 선택 가능
- 프로그램당 최대 정원: 7명
- 제약사항: Google 생태계 내에서만 구현

## 2. 시스템 구조

### 2.1 데이터 레이어 (Google Sheets)

#### 기존 데이터
- **멘토 프로그램 목록** (Google Forms 응답 시트)
  - 멘토가 Google Forms로 제출한 프로그램 정보
  - 컬럼 구조: result table.csv 참조

#### 새로 생성할 데이터
- **신청현황 시트**
  - 멘티들의 프로그램 신청 기록
  - 컬럼: 신청ID, 멘티학번, 멘티이름, 멘티이메일, 프로그램ID, 신청일시, 상태, 개인정보동의여부

- **설정 시트**
  - 시스템 운영에 필요한 설정값
  - 신청 기간, 최대 신청 수, 정원 등

### 2.2 비즈니스 로직 (Google Apps Script)

#### 핵심 기능
1. **신청 유효성 검사**
   - 멘티당 최대 3개 프로그램 제한
   - 동일 프로그램 중복 신청 방지
   - 프로그램별 7명 정원 체크
   - 시간표 충돌 검사 (같은 멘티의 프로그램끼리)

2. **데이터 연동**
   - Forms 응답 시트에서 프로그램 목록 읽기
   - 신청 정보를 신청현황 시트에 기록
   - 실시간 신청 인원 업데이트

3. **자동화 기능**
   - 정원 마감 시 자동 상태 변경
   - 신청 확인 이메일 발송

### 2.3 프레젠테이션 레이어 (웹 인터페이스)

#### 멘티용 신청 페이지
- 프로그램 목록 표시 (카드 또는 테이블 형식)
- 필터링 기능 (날짜, 시간, 멘토 등)
- 신청 버튼 및 신청 현황 표시
- 본인 신청 내역 조회

#### 멘토용 조회 페이지
- 자신의 프로그램별 신청자 명단 조회
- 신청자 연락처 확인

#### 관리자용 대시보드 (선택사항)
- 전체 신청 현황 통계
- 프로그램별 신청률

## 3. 캘린더 연동 방안 (Option B: 공유 캘린더)

### 구현 방식
1. "2025 신입생 멘토링 프로그램" 공유 캘린더 생성
2. 신청 마감 후 확정된 모든 프로그램 일정 등록
3. 일정 제목: 프로그램명
4. 일정 설명: 프로그램 내용, 멘토 정보, 참가 멘티 명단
5. 멘토와 멘티는 공유 캘린더 구독으로 일정 확인

### 장점
- 권한 문제 해결
- 중앙 집중식 관리
- 모든 참가자가 전체 프로그램 일정 확인 가능

## 4. 구현 우선순위

### Phase 1 (필수 - 9월 8일 전 완료)
1. Forms 응답 데이터 구조 파악
2. 신청현황 관리 시트 생성
3. 멘티 신청 웹앱 개발
   - 프로그램 목록 표시
   - 시간 충돌 검사
   - 신청 처리 로직

### Phase 2 (필수 - 9월 12일 전 완료)
1. 조회 기능 구현
   - 멘토: 신청자 명단 조회
   - 멘티: 신청 내역 확인
2. 신청 마감 처리 로직

### Phase 3 (선택 - 9월 15일 전 완료)
1. 공유 캘린더 생성
2. 확정 일정 자동 등록 스크립트
3. 참가자 안내 이메일 발송

## 5. 기술 스택
- **데이터베이스**: Google Sheets
- **백엔드**: Google Apps Script (JavaScript)
- **프론트엔드**: HTML, CSS, JavaScript
- **캘린더**: Google Calendar API
- **배포**: Google Apps Script Web App

## 6. 주의사항
- 모든 구현은 Google 생태계 내에서 진행
- 코드는 단순하고 유지보수가 쉽도록 작성
- 매뉴얼은 별도 문서로 작성
- 학번 인증은 별도로 구현하지 않음
